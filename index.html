Troubleshooting Guide
Low biogas production +

Check the pH level (should be 6.8-7.2) and temperature (35-40°C). Ensure adequate organic matter and proper mixing.

If production remains low, try activating the biocatalyst or adjusting the C:N ratio by adding more carbon-rich materials.

Biocatalyst not activating +

Check if the catalyst reservoir has sufficient biological catalyst remaining. Refill if below 20%.

Verify auto-activation settings and ensure the system is in production mode. Contact support if issues persist.

Pressure sensor error +

Reset the device and check the pressure sensor connections. Clean the sensor if needed.

If the error persists, the sensor may need replacement. Contact technical support for assistance.

Poor gas quality (low methane) +

This usually indicates incomplete digestion. Check temperature consistency and ensure proper pH levels.

Activate the biocatalyst to enhance microbial activity and improve methane production efficiency.
