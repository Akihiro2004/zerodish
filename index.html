Troubleshooting Guide
Device won't turn on +

Ensure the power cable is properly connected and the outlet is functional. Also, check if the power button on the back of the device is in the "ON" position.

If it still doesn't turn on, contact technical support.

Process stopped midway +

This can occur if the humidity sensor detects abnormal conditions. Open the device and check for any foreign objects in the chamber.

If the issue persists, try resetting the device by pressing the reset button for 5 seconds.

Sensor error +

Reset the device and check the sensor connections. If the error persists, the sensor may need replacement. Contact technical support for assistance.
